# Responses

## Responses to One of a Major (1H / 1S)

### Point Ranges for Responder

| Strength | Points | Action |
|----------|--------|--------|
| Minimum | 6–9 (10) | Simple raise, 1-level new suit, 1NT |
| Invitational | 10–12 | Limit raise, new suit, 2-over-1 |
| Game forcing | 13+ | Jacoby 2NT, new suit, jump shift |
| Slam interest | 19+ | Jump shift |

### Specific Responses

| Response | Requirements |
|----------|-------------|
| New suit at 1-level (e.g., 1H→1S) | 4+ cards, 6+ points; forcing one round by unpassed hand |
| 1NT | 6–10 HCP; denies 3-card support for opener's major; denies 4 spades over 1H; non-forcing |
| 2C / 2D (2-over-1) | 4+ cards, 10+ points (some sources say 11+); forcing one round; promises a rebid unless opener jumps to game |
| 2H (over 1H) or 2S (over 1S) — single raise | 3+ card support, 6–10 dummy points |
| 3H / 3S — limit raise | 3+ card support, 10–12 dummy points; invitational |
| 2NT — Jacoby 2NT | 4+ card support, 13+ dummy points; game forcing; asks opener to describe hand (see [05-conventions.md](05-conventions.md)) |
| 3NT | 15–17 HCP, balanced, exactly 2-card support; to play |
| 4H / 4S — preemptive raise | 5+ card support, singleton or void, fewer than 10 HCP |
| Jump shift (e.g., 1H→2S, 1H→3C) | 19+ points, 4+ cards in new suit (5+ for minor jump shifts); slam invitational |

### Support Points (Dummy Points)

When raising partner's major, add shortness points to HCP:

| Holding | Add |
|---------|-----|
| Void | 5 |
| Singleton | 3 |
| Doubleton | 1 |

Do not count side-suit honors as both HCP and support points.

## Responses to One of a Minor (1C / 1D)

### Trump Support Requirements

- **Raise 1D**: 4+ card support required
- **Raise 1C**: 5+ card support required (or 4 in competition)
- There is **no forcing minor-suit raise** in SAYC

### Specific Responses

| Response | Requirements |
|----------|-------------|
| New suit at 1-level | 4+ cards, 6+ points; bid up the line with multiple 4-card suits; forcing one round |
| 1NT | 6–10 HCP, no 4-card major; non-forcing |
| Single raise (2C/2D) | Adequate trump support (see above), 6–10 points |
| Jump raise (3C/3D) — limit raise | Adequate trump support, 10–12 points; invitational |
| 2NT | 13–15 HCP, balanced, denies 4-card major; game forcing |
| 3NT | 16–17 HCP, balanced, denies 4-card major; to play |
| Jump shift (e.g., 1C→2H) | 19+ points, 5+ cards; slam invitational |
| 2-over-1 in new suit (e.g., 1D→2C) | 4+ cards, 10+ points; forcing one round |

### Priority for Responding

1. Show a 4-card major at the 1-level
2. If no 4-card major, raise partner's minor with adequate support
3. If neither, bid 1NT

**Note**: Do not add support points for shortness when raising a minor — the final contract may be notrump.

## Responses to 1NT (15–17)

| Response | Requirements |
|----------|-------------|
| Pass | 0–7 HCP, balanced |
| 2C — Stayman | 8+ HCP (or weaker with 4-4-4-1 shape); asks for 4-card major; non-forcing (see [05-conventions.md](05-conventions.md)) |
| 2D — Jacoby transfer | 5+ hearts, any strength; requests opener bid 2H (see [05-conventions.md](05-conventions.md)) |
| 2H — Jacoby transfer | 5+ spades, any strength; requests opener bid 2S |
| 2S — minor suit puppet | Requests opener bid 3C; responder passes (club bust) or bids 3D (diamond bust) |
| 2NT | 8–9 HCP, balanced; invitational to 3NT |
| 3C / 3D | 6+ card suit, invitational to 3NT; typically 2 of top 3 honors in suit |
| 3H / 3S | 6+ card suit, slam interest |
| 3NT | 10–15 HCP, balanced; to play |
| 4C — Gerber | Ace-asking (see [06-slam.md](06-slam.md)) |
| 4D / 4H — Texas transfer | 6+ card major, sign-off at game (if played; not all SAYC pairs use Texas) |
| 4NT | Quantitative; invites 6NT (opener passes with minimum, bids 6NT with maximum) |

## Responses to 2C (Strong, Artificial, Forcing)

| Response | Requirements |
|----------|-------------|
| 2D — waiting | Artificial; may be made with any hand (including a good hand waiting to hear opener's suit) |
| 2H / 2S / 3C / 3D — natural positive | 5+ card suit, typically 8+ HCP, 2 of top 3 honors; natural and game forcing |
| 2NT | Balanced, 8+ HCP |

### After Opener Rebids 2NT (22–24 balanced)

All standard 2NT responses apply: 3C Stayman, 3D/3H transfers, 4C Gerber, 4NT quantitative.

### Forcing Rules

- After a positive response: game forcing
- After 2D waiting and opener bids a suit: forcing to 3 of a major or 4 of a minor
- After 2D waiting and opener rebids 2NT: not forcing (responder may pass with a very weak hand)

## Responses to Weak Two-Bids (2D / 2H / 2S)

| Response | Requirements |
|----------|-------------|
| Raise (3H, 3S, 3D) | To play; often preemptive; does not show extras |
| 2NT — feature ask | Forcing (even when opponents intervene); game interest |
| 3NT | To play |
| New suit | 5+ cards; forcing one round (RONF — Raise Only Non-Force) |
| Jump raise to game (4H, 4S) | To play; may be preemptive with good support |

### Opener's Rebid After 2NT Feature Ask

| Opener's Hand | Rebid |
|---------------|-------|
| Minimum (5–8 HCP) | Rebid own suit at 3-level |
| Maximum with feature | Show feature (outside ace or protected king) |
| Maximum, no feature | Bid 3NT |

### Opener's Rebid After New Suit Response

| Condition | Rebid |
|-----------|-------|
| 3+ card support for responder's major | Raise |
| No fit, minimum | Rebid own suit at cheapest level |
| No fit, maximum | New suit or notrump |

| Response | Requirements |
|----------|-------------|
| 3C — Stayman | Asks for 4-card major |
| 3D — Jacoby transfer | 5+ hearts |
| 3H — Jacoby transfer | 5+ spades |
| 3S | Minor suit puppet (to 4C, then pass or bid 4D) |
| 3NT | To play |
| 4C — Gerber | Ace-asking |
| 4NT | Quantitative; invites 6NT |

## Responses to 3-Level Preempts (3C / 3D / 3H / 3S)

| Response | Requirements |
|----------|-------------|
| Raise | Preemptive; to play |
| 3NT | To play |
| New suit below game | 5+ cards; forcing one round |
| Jump raise to game | To play |

Responder should have a source of tricks and/or stoppers for 3NT. New suits are forcing for one round.
